
# 🎟️ Event Hub Booking System

## 📌 Project Description
A simple, responsive event booking web application that allows users to browse and book events with real-time notifications. Built for modern web development practice and CS portfolio showcase.

**Problem Statement**:  
Event organizers need an easy-to-use platform for users to discover upcoming events and complete bookings seamlessly without complex backend setup.

---

## ✨ Features
- Browse available events with details
- Real-time booking confirmations
- Responsive design for all devices
- Clean, modern user interface

---

## 🛠️ Technology Used
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Tools**: VS Code, Git/GitHub, Live Server
- **Styling**: CSS Grid, Flexbox, Animations

## 🚀 Usage

1. Open "HOME.html" in your browser
2. Browse available events
3. Click **"Book Now"** on any event
4. View booking confirmation with event details
